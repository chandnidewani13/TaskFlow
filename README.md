
# 🚀 TaskFlow

AI-Ready Team Productivity Management System

TaskFlow is a web-based productivity and project management platform that helps organizations efficiently manage projects, teams, tasks, meetings, and reports. The system provides separate dashboards for Admin, Business, Team Leader, and Employee users with role-based access control.

---

## 📌 Features

### 👨‍💼 Admin
- User Management
- Team Management
- Project Management
- Analytics Dashboard
- Reports
- Notifications
- File Uploads
- Approval Management

### 🏢 Business
- Business Dashboard
